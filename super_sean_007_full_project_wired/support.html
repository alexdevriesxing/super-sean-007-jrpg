Support and troubleshooting

Most problems can be fixed without losing progress. Back up your save code before clearing browser data or changing devices.

Return to the game

Game does not start

Reload the page, disable aggressive script blockers for this site, and use a current version of Chrome, Edge, Firefox or Safari. Private browsing may restrict storage and service-worker behavior.

Save or Cloud Sync problem

Open Settings and download a JSON backup or copy the save code before troubleshooting. Keep your Cloud Sync ID private; it acts like a password. The local save remains separate from the cloud copy.

No sound

Browsers require a click, key press or controller action before audio can begin. Check the music and sound sliders in Settings and confirm the browser tab is not muted.

Controller not detected

Connect the controller before starting the game, press a button, and confirm Gamepad input is enabled in Settings. Wired or officially supported Bluetooth controllers are the most reliable.

Party Link cannot connect

Try again on a different network and confirm both players use the current release. Strict corporate networks and some mobile carriers may require the configured TURN relay.

Ads or optional rewards unavailable

Third-party units load only after advertising consent. Ad blockers, regional inventory, network filtering or declined consent can make optional ad-funded rewards unavailable without blocking normal play.

Before reporting a bug

Include reproducible details

  • Device, operating system and browser version.
  • The game version shown on Service Status.
  • The map, quest or battle where the problem occurred.
  • Exact steps that reproduce the problem.
  • A screenshot, but never publish a Cloud Sync ID or save code.

Open the bug-report form

Security vulnerabilities must be reported through the Security Policy, not a public issue.
