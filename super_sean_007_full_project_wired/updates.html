13 July 2026 · Production and discovery update

Safer APIs, fresh deployments and expanded game guides

The deployment has been hardened around four priorities: protecting diagnostics, preventing stale game assets, returning real 404 responses and keeping public game facts consistent.

  • Analytics and error-log reads now require a private Cloudflare ADMIN_TOKEN.
  • API writes are restricted to approved origins and protected with hashed, short-lived rate-limit keys.
  • Error reports no longer store user-agent strings.
  • Mutable assets use revalidation instead of unsafe year-long immutable caching.
  • The service-worker cache version is generated from each deployment commit.
  • Unknown routes return a branded 404 instead of a duplicate homepage.
  • Guides, characters, world and update pages provide independently indexable, useful content.
  • A canonical site-facts file updates the homepage, sitemap and machine-readable summaries during every build.
  • GitHub Actions now builds and validates every proposed change.

12 July 2026 · World and visual expansion

Frostpeak Reaches, Sunsand Isle and the final asset-wiring round

Two postgame regions were added after the main Xelar confrontation. Frostpeak Reaches introduces elite demons and the Void Succubus Queen; defeating her opens Sunsand Isle and its Tide Spirit Sovereign encounter.

The same release added party allies to battle scenes, battle-start and victory cards, status-effect icons, additional dark knights and minions, more village NPCs, landmark buildings, ambient animals and expanded visual effects. Heavy assets continue loading in controlled batches so mobile boot is not held hostage by a single slow image.

11 July 2026 · Online and persistence systems

Cloud Sync, Party Link and first-party analytics

Players can keep saves locally, move them with copyable save codes or voluntarily enable anonymous Cloud Sync. Party Link uses Cloudflare only for the short WebRTC handshake, after which supported gameplay traffic is peer to peer. Cookieless first-party analytics counts aggregate gameplay milestones without requiring an account.

9–10 July 2026 · Original launch foundation

The free browser JRPG goes live

The first public foundation combined a fifteen-chapter campaign, the Seven Gems story, turn-based party battles, nine opening regions, harvesting, nineteen crafting recipes, equipment, gardening and a tile-based homestead with blueprints and Comfort perks.
